@media queries

CSS uses the @media rule to detect information about the decive being used to accsess the website. For example, it can be used to detect the size, its capabilities, and it orientation. For this course, we will be using it to detect size, but keep in mind that it is capable of much more.

Desktop: Widdth greater than or equal to 1024px

Tablet: Width between 481px and 1023px

Phone: Width les than or equal to 480px
